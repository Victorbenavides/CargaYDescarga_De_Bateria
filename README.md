# CargaYDescarga_De_Bateria
Aqui estamos graficando las curvas de carga y descarga de una batería para analizar su rendimiento a través de múltiples ciclos de uso. Primero procesamos los datos para identificar en qué ciclo se alcanzó el tiempo máximo tanto de carga como de descarga, convirtiendo esos valores de segundos a un formato de horas y días más fácil de leer. Después generamos una gráfica lineal donde la línea azul representa el tiempo de carga y la roja el de descarga, añadiendo etiquetas y flechas automáticas que señalan los puntos más altos de operatividad. Finalmente, configuramos el diseño visual con cuadrículas y límites de ejes para guardar la imagen como battery_curves.png, lo que nos permite visualizar rápidamente las anomalías o picos de duración en la vida útil de la batería.
